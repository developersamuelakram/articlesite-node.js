# Simple Articles Site built with Node.js - Mongoose - Express and Bootstrap


1. Bootstrap ✔
2. PUG ✔
3. Express ✔
4. Add, Edit, Delete Articles ✔
5. Mongo and Mongoose integerated ✔
6. JSON response API ✔ (/api/jsonposts)
